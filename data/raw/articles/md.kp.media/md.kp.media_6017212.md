# В Молдове автомобиль сбил 14-летнюю школьницу прямо на пешеходном переходе: Она переходила дорогу по правилам

Автомобиль сбил школьницу на пешеходном переходе. Фото: соцсети.

Авария произошла 29 сентября на центральной улице Фдлоешт.

По данным полиции, 14-летняя девочка предварительно слезла с электросамоката и переходила дорогу по правилам - по пешеходному переходу. В этот момент её сбил автомобиль Suzuki, за рулём которого находился 64-летний водитель, передаёт tvn.md.

Школьницу госпитализировали. Обстоятельства ДТП выясняются.
