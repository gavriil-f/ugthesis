# Команда из Гагаузии “Трудные Подростки” выиграла в Чемпионате по интеллектуальным играм

На Республиканском Чемпионате по интеллектуальным играм “Sub cuşma lui Guguţa” победила команда «Трудные Подростки» из Гагаузии.

По итогам отборочного этапа Чемпионата, из 511 команд в финал вышли 26 команд, 6 из которых оказались выходцами из гагаузских учебных заведений.

Среди Лицейского цикла Гагаузию представляли команды:

“Трудные Подростки” из Лицея им. м. Губогло мун. Чадыр-Лунга“Копчакские Интеллектуалы” из Лицея им. Б. Янакогло с. Копчак“Поколение Z” из Лицея им. В. Мошкова мун. Чадыр-Лунга.

- “Трудные Подростки” из Лицея им. м. Губогло мун. Чадыр-Лунга
- “Копчакские Интеллектуалы” из Лицея им. Б. Янакогло с. Копчак
- “Поколение Z” из Лицея им. В. Мошкова мун. Чадыр-Лунга.

Среди Гимназического цикла Гагаузию представляли команды:

“Эврика” из Лицея им. м. Губогло мун. Чадыр-Лунга“Копчакские Интеллектуалы” из Лицея им. Б. Янакогло с. Копчак“Многоточие” из Лицея им С. Барановского с. Копчак.

- “Эврика” из Лицея им. м. Губогло мун. Чадыр-Лунга
- “Копчакские Интеллектуалы” из Лицея им. Б. Янакогло с. Копчак
- “Многоточие” из Лицея им С. Барановского с. Копчак.

Победителем и Чемпионом в этой интеллектуальной битве среди Лицейского цикла стала команда из Гагаузии — “Трудные Подростки” из Лицея им. м. Губогло мун. Чадыр-Лунга.
