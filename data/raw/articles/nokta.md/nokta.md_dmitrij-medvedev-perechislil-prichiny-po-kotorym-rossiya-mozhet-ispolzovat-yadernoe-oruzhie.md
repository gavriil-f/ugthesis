# Дмитрий Медведев перечислил причины, по которым Россия может использовать ядерное оружие

Зампред Совета Безопасности РФ Дмитрий Медведев перечислил четыре основания, на которых Россия может применить ядерное оружие. При этом, по словам Медведева, Россия применит ядерное оружие только для «самообороны».

В интервью французскому телеканалу LCI Медведевперечислил4 причины применения ядерного оружия Россией.

Старт ядерных ракет [противника],применение [им] ядерного оружия,нанесение удара по критической инфраструктуре, которая управляет ядерным оружием,или иные действия, которые грозят существованию самого государства [РФ].

- Старт ядерных ракет [противника],
- применение [им] ядерного оружия,
- нанесение удара по критической инфраструктуре, которая управляет ядерным оружием,
- или иные действия, которые грозят существованию самого государства [РФ].
