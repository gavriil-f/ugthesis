# Инициативная группа по организации референдума по отмене смешанной избирательной системы намерена снова добиваться регистрации в ЦИК

Инициативная группа по организации референдума по отмене смешанной избирательной системы будет продолжать добиваться регистрации своей инициативы в ЦИК Молдовы. Об этом сообщил глава инициативной группы Штефан Глигор во время пресс-конференции 16 декабря.

По его словам, рассматриваются два пути: создать новую инициативную группу и заново подать документы в ЦИК либо оспорить в суде действия ЦИК по отказу в регистрации инициативы по проведению референдума по отмене смешанной избирательной системы.

## Что произошло

В пятницу, 12 декабря ЦИКотклонилрегистрацию инициативной группы по организации республиканского законодательного референдума для отмены смешанной избирательной системы по причине «несоответствия требованиям законодательства».

Представители гражданского обществасоздалиинициативную группу по отмене закона о переходе к смешанной избирательной системе 17 декабря 2017 года.

Председатель инициативной группы Штефан Глигор сказал, что референдум по отмене смешанной избирательной системы необходимо провести, чтобы «исправить злоупотребление, совершённое Демократической партией при помощи социалистов».
