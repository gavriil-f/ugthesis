# Действующие подъездные пути к избирательным участкам для избирателей в приднестровском регионе

Бюро политик по реинтеграцииинформируеторганы местного публичного управления и заинтересованные стороны о списке маршрутов для доступа к местам, где будут открыты избирательные участки для граждан Республики Молдова из приднестровского региона.

Ведомство напоминает, что к президентским выборам 1 ноября в 22 населенных пунктах на правом берегу Днестра и в Дубоссарском районе будут открыты 42 избирательных участка, предназначенных для граждан с правом голоса, проживающих в административно-территориальных единицах на левом берегу Днестра. и мун. Бендер.

Список действующих дорог/подъездных дорог представлен ниже:

Город Анений Ной — R2 Кишинев — Бендеры / М5;

Коммуна Калфа — R2 Кишинев — Бендеры / M5;

Село Гура Быкулуй — мост через Днестр Бычок — Гура Бакулуй (M5);

Село Хырбовэц — подъездная дорога L476 R2;

Село Варница — L475 подъездная дорога Варница — Гура Бакулуй;

Город Каушаны — R26 Бендеры — Каушаны / G113 Копанка — Плоп-Штюбей — Каушаны;

Село Киркэйешть — паромная переправа (Тирасполь — Кицканы — Киркэйешть) / L569 Бендеры — Киркэйешть — R26;

Село Копанка — подъездная дорога G113 (Бендеры — Кицканы — Каушены) / паромная переправа L581 (Слобозия-Кременчуг-Копанка);
