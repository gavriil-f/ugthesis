# Елена Кобэсняну избрана председателем Ассоциации независимой прессы

Директор Сорокской газеты «Северная обсерватория» Елена Кобэсняну была избрана председателем Ассоциации независимой прессы (API) на последующие два года.

Решение было принято членами API на Генеральном собрании, которое состоялось 12 декабря 2021 года в режиме онлайн.

Генеральное собрание было созвано после отставки с должности председателя API Еуджена Рыбка, который был назначен парламентом в качестве члена Совета по телевидению и радио по рекомендации API и Ассоциации Media-Guard.

На Генеральном собрании был подтвержден обновленный состав административного совета API, в который войдут, помимо председателя API Елена Кобэсняну и вице-председатель API Елена Мотрикалэ (директор газеты” Est Curieră » в Криулянах), еще пять представителей и представительниц/учреждений независимых СМИ. Речь идет об Алине Раду (директор „Ziarul de Gardă”), Славе Перунов (директор портала Esp.md из Бельц), Виктории Попа (директор «Газеты Кишинев»), Михаиле Сиркели (директор портала Nokta.md из Комрата) и фотожурналист Константин Григорицэ.

Елена Кобэсняну уже более 23-х лет руководит газетой „Северная обсерватория” и занимала еще один мандат председателя API в период с 2004 по 2006 год.

Ассоциация независимой прессы (API) была создана в 1997 году для содействия развитию независимой прессы в Республике Молдова.
