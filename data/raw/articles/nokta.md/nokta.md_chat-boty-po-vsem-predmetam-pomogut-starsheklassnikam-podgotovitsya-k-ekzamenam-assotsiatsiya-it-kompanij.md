# Чат-боты по всем предметам помогут старшеклассникам подготовиться к экзаменам — ассоциация IT-компаний

В Молдове для выпускников гимназий и лицеистов специалисты сферы информационных технологий Ассоциации ATIC подготовили электронных репетиторов. В первый день занятий они поделились ссылками на чат-боты, призванные помогать учащимся 9-12 классов в изучении новых тем и проверке своих знаний.

Всем ученикам 9-12 классов по случаю начала учебного года председатель Национальной ассоциации компаний в сфере информационно-коммуникационных технологий ATIC Вячеслав Куневподелился списком ссылокна чат-боты по всем предметам бакалавриата на русском и румынском языках.

Эти ресурсы можно использовать:

Когда необходима дополнительная помощь по какой-либо теме во время учебного процесса.Для тестирования своих знаний.Для проверки своей готовности к сдаче экзаменов по отдельным предметам.

- Когда необходима дополнительная помощь по какой-либо теме во время учебного процесса.
- Для тестирования своих знаний.
- Для проверки своей готовности к сдаче экзаменов по отдельным предметам.

Чат-боты будут ежегодно обновляться в соответствии с обновлениями учебных программ. Чтобы воспользоваться чат-ботом нужноиметь доступ к бесплатной или платной версии ChatGPT, уточнил Кунев. Каждый чат-бот содержит материалы 9-12 классов по соответствующему предмету.
