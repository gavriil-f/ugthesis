# Как заполнить бюллетени для голосования по почте — инструкция ЦИК

Гражданам Молдовы, участвующим в голосовании по почте и уже получившим конверт с бюллетенями, ЦИК рекомендует для начала ознакомиться с инструкцией по их заполнению.

Центральная избирательная комиссия просит граждан, которые приняли решение участвовать по почте в голосовании на выборах президента Молдовы и еврореферендуме, внимательно прочитать  инструкцию по заполнению бюллетеня для голосования,сообщаетМинистерство иностранных дел Молдовы. Лишь после этого можно приступать к его заполнению.
